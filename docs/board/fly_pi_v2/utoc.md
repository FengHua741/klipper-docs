#  10. 板载UTOC的使用

## 10.1 拨码开关设置

启用板载**UTOC**，需将下图所示的两个拨码开关向上拨至ON状态！！！默认为关闭状态！！！

![utoc1](../../images/boards/fly_pi_v2/utoc1.jpg)

> [!WARNING]
> 使用拨码开关启用板载**UTOC**后，下图所示的USB口会被占用，不能再接入任何设备！！！

![utoc2](../../images/boards/fly_pi_v2/utoc2.jpg)
