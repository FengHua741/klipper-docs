>[!TIP]
>FLY系统镜像在M2WE和SD卡都有系统情况下默认启动SD卡系统

>[!Warning]
>SD卡注意事项:
>容量必须不小于16GB 或 不大于128G TF 卡
>TF 卡的传输速度必须为 class10 级或 class10 级以上，建议使用闪迪等品牌的 TF 卡

1. SD卡一张

2. 读卡器

3. 系统镜像

4. 工具软件请前往FLY官方群文件夹下载：Fly-Gemini系列； 需要用到的软件有：balenaEtcher（fly群中文件名为系统镜像写入工具）、putty、WinSCP
